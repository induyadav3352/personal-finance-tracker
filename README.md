# Personal Finance Tracker

A Python-based application to manage personal income and expenses.

## Features
- Add Income
- Add Expenses
- View Transactions
- Financial Summary
- Category-wise Expense Analysis
- Expense Visualization using Matplotlib
- Delete Transactions

## Technologies Used
- Python
- Pandas
- Matplotlib
- CSV

## Installation

pip install -r requirements.txt

## Run

python finance_tracker.py
